# laravel Project

This is a simple application CRUD built with Laravel and Tailwind CSS.

## Features

- Auth User
- Add Product
- Edit Product
- Delete Product
- Search Product

## Technologies Used

- Laravel
- Tailwind CSS
- Alpine Js
